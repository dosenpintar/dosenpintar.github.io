14. Jika akar-akar persamaan kuadrat 2x²-x+4=0 adalah a dan B, maka Persamaan kuadrat yang akar-akarnya (a-2) dan (b-2) adalahplis bantu jawab soalnya besok dikumpul​


Penjelasan dengan langkah-langkah:

PERSAMAAN KUADRAT (PK)

Diketahui:

PK : 2x²-x+4 = 0

dengan akar²nya a dan b

Ditanya:

PK baru dengan akar² (a-2) dan (b-2) adalah..

Jawab:

2x²-x+4=0

=>

a+b = -(-1) /2 = 1/2

a. b = 4/2 = 2

untuk akar² (a-2) dan (b-2) :

hasil penjumlahan nya:

(a-2) +(b-2) = a-2+b-2

= a+b -4

= 1/2 -4 = -3½

hasil perkalian nya:

(a-2)(b-2) = ab - 2a-2b+4

= a.b -2(a+b) +4

= 2 -2(½) +4

= 2-1+4

= 5

maka PK barunya :

x²-(3½) x +5 = 0

=>

2x² -7x +10 = 0

~Persamaan Kuadrat

____________________

» Pembahasan

2x² - x + 4 = 0

2x² - 1x + 4 = 0

didapat:

a = 2b = - 1c = 4

• Menentukan a + b

= a + b

= - b/a

= - ( - 1 )/2

= ½

• Menentukan a × b

= a × b

= c/a

= 4/2

= 2

• Menentukan akar-akar persaman baru

Hasil jumlah akar

= (a - 2) + (b - 2)

= (a + b) - (2 + 2)

= a + b - 4

= ½ - 4

= 0,5 - 4

= - 3,5

= - 7/2

Hasil kali akar

= (a - 2)(b - 2)

= (a × b) + (a × ( - 2 )) + (b × ( - 2 )) + (( - 2 ) × ( - 2 ))

= ab + ( - 2a ) + ( - 2b ) + 4

= ab - 2a - 2b + 4

= ab - 2(a + b) + 4

= 2 - 2(½) + 4

= 2 - 1 + 4

= 1 + 4

= 5

Sehingga persamaan kuadrat baru yang akar-akarnya (a - 2) dan (b - 2) diperoleh:

x² - ((a - 2) + (b - 2))x + ((a - 2)(b - 2)) = 0

x² - (7/2)x + 5 = 0

agar tidak berbentuk pecahan, kalikan kedua ruas dengan 2 diperoleh:

2x² - 7x + 10 = 0
