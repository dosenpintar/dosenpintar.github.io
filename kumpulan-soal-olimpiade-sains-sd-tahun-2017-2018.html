Perhatikan data banyak siswa SD yang mengikuti olimpiade matematika tingkat kecamatan selama 4 tahun terakhir pada diagram lingkaran berikut!

2016 = 60°

2017 = 30°

2019 = 180°

2018 = 90°

Jika banyak peserta olimpiade pada tahun 2017 ada 19 orang, banyak peserta pada  tahun 2019 adalah 114 orang​

A. 58 orang

B. 87 orang

C. 174 orang

D. 114 orang​

Penjelasan:

Satu lingkaran penuh adalah 180°, dan jika pada tahun 2017 atau 30°  sama dengan 19 0rang maka 180°  adalah

180° / 30°  = 6

6 x 19 = 144

Jadi jumlah peserta olimpiade pada tahun 2019 adalah sebanyak 114 orang.

Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
