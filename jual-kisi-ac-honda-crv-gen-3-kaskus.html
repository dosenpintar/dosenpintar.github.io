8. dijual tunai sebuah sepeda motor Honda seharga Rp 18.000.000,00TERMASUK JURNAL APA​


Jawaban: Harga awal = 18000000

Bunga = 10% per tahun

Uang muka = 6000000

Angsur selama = 4 tahun

Bunga selama 4 tahun = 4 x 10% = 40%

Sisa pembayaran(belum +bunga)

= harga awal - uang muka

= 18000000 - 6000000

= 12000000

Sisa pembayaran dengan bunga (selama 4 tahun)

= sisa pembayaran + (bunga x sisa pembayaran)

= 12000000 + (40% x 12000000)

= 12000000 + 4800000

= 16800000

Angsuran perbulan

= 16800000/waktu angsur

= 16800000/48 bulan

= 350000/bulan

Jadi, harus membayar angsuran Rp.350000 perbulannya

38. Jumlah usia honda dan bambang adalah 93. Jika usia honda 3 tahun yang lalu sama dengan usia bambang 8 tahun dari sekarang, berapa usia honda sekarang? sepi...


Usia Honda sekarang adalah 52 tahun

PENJELASAN

Misal:

Usia Honda = xUsia Bambang = y

Diketahui:

Jumlah usia x dan y adalah 93Usia x - 3 = y + 8

Ditanyakan:

Usia x

Jawab:

Jumlah x dan y adalah 93

=> x + y = 93  (I)

x - 3 = y + 8 (II)

=> x = y + 8 + 3

=> x = y + 11 (III)

Substitusikan III ke I

x + y = 93

y + 11 + y = 93

2y + 11 = 93

2y = 93 - 11

2y = 82

y = 82/2

y = 41

Nilai y (usia Bambang) adalah 41

Substitusikan nilai y ke III

x = y + 11

x = 41 + 11

x = 52

Nilai x (usia Honda) adalah 52

PEMBUKTIAN

x + y = 93

52 + 41 = 93

93 = 93  (Benar)

x - 3 = y + 8

52 - 3 = 41 + 8

49 = 49  (Benar)

KESIMPULAN

Usia Honda sekarang adalah 52 tahun
