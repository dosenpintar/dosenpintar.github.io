32. type 2 dan type 3 beserta fact


rumus if type 2:
*if + S+ VERB 2/WERE, S+ would+ VERB 1
*fact: S+VERB1/ VERB 1+-s/es (simple present)
1. If they listened carefully,they would hear yhe singing bird.
2. If I laid down on the bed, I would fall asleep
fact: i don't lay down so don't fall asleep.


rumus type3:
*if +S+had+VERB 3, S + would have+ VERB 3
*fact: S+VERB 2(simple past)
1. Diana would have worked as a model if she had been taller.
fact: Diana didn't work as a model because wasn't tall.
2. The soil wouln't have dried out if you had watered the plants regularly.
fact: the soil dried out because you didn't water the plants regularly.
3. if the cat had hidden in the tree, the dog wouldn't have found it.
fact: the cat didn't hide in the tree so the dog found it.

39. buatlah rumus Conditional if ( semua type) 1.2,3 beserta contoh kalimatnya


Jawaban:

0 Conditional

Form: If + present simple, .... present simple

Example: If you touch a fire, you get burned.

1st Conditional

Form: if + Simple Present, will-Future

Example: If I find her address, I’ll send her an invitation.

2nd Conditional

Form: if + Simple Past, Conditional I (= would + Infinitive)

Example: If I found her address, I would send her an invitation.

3rd Conditional

Form: if + Past Perfect, Conditional II (= would + have + Past Participle)

Example: If I had found her address, I would have sent her an invitation.
