Penjelasan:

Simple present tense

( + ) S + V1 + O + Adverb of time

( - ) S + do'nt/does'nt + V1 + O + Adverb of time

( ? ) do'nt/does'nt + S + V1 + O + Adverb of time + ?

Keterangan :

Subject :  I, You, They, We, She, He, It, People name

Verb 1 : Cook, Eat, Run, etc

Object : Table, Chair, place name, etc

Adverb of tense : Every day, every morning, every month, every year, etc

Example : Taqwim eat rice in the table every day

Simple present tense digunakan untuk kegiatan sehari-hari.

Simple continous tense

( + ) S + be ( is,am,are ) + V1 + ing + O + Adverb of time

( - ) S +  be ( is,am,are ) + Did'nt + V1 + ing + O + Adverb of time

( ? ) Did'nt + S +  be ( is,am,are ) +  V1 + ing + O + Adverb of time + ?

Keterangan :

Subject :  I, You, They, We, She, He, It, People name

Verb 1 + ing : Cooking, Eating, Running, etc

Object : Table, Chair, place name, etc

Adverb of time : Now, Today, Tonight, etc

Example : Taqwim is eat on the table now ( + )

(-) S + Do/Does + not + V1

(+) S + tobe (is, am.are) + V1 (ing)

(+) S + V2
(-) S + did not + V1
(?) Did + S + V1?

Simple future
(+) S + will/shall + V1
(-) S + will/shall + not + V1
(?) will/shall + S + V1 ?Simple Present: S+V1+O

Present Perfect: S+have/has+V3+O

Simple Past: S+V2+O

Past Perfect: S+had+V3+O
Past Perfect Continuous: S+had+been+V1ing+O

Simple Future: S+shall/will+V1+O
Future Continuous: S+shall/will+be+V1ing+O
Future Perfect: S+shall/will+have+V3+O
Future Perfect Continuous: S+shall/will+have+been+V1ing+O

Note:
shall= I, we
will=you,they,he,she,it
was=I,he,she,it
were=you,they,we
am=I
is=he,she,it
are=you,we,they

Simple Present Tense ( Waktu sekarang )
contoh :
(+) : Icha is a teacher (icha seorang guru)
(-)  : Icha is not a teacher (icha bukan seorang guru)
(?) : Is Icha a teacher (apakah icha seorang guru)
(-?): Isn't Icha a teacher ? (tidaklah icha seorang guru)
Simple Perfect Tense( Waktu Sempurna Sekarang )
keterangan waktu yg sering dipakai adalah :
For(selama), Since (sejak), Already (telah) , Ever (pernah), Never(tidak pernah)
(+) : I have already eaten ( saya telah makan )
(-)  : I haven't already eaten ( saya belum selesai makan )
(?) : Have I already eaten ? ( apakah saya telah makan ? )
(-?): Haven't already eaten ? ( tidaklah saya telah makan ? )

PEMBAHASAN MATERI

ʕ Simple Past Tense ʔ

ᴥ Fungsi

Mengungkapkan suatu peristiwa/kegiatan yang terjadi atau dilakukan di masa yang telah berlalu.

ᴥ Pattern:

(+) Subject + Verb 2 + Object/Complement(–) Subject + didn't + Verb 1 + Object/Complement(?) Did + Subject + Verb 1 + Object/Complement + ?

◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌

ʕ Simple Past Continuous ʔ

ᴥ Fungsi

Mengungkapkan suatu peristiwa/kegiatan yang sedang terjadi atau berlangsung di masa yang telah berlalu. Misalnya: Saya sedang menulis berita saat kamu datang kemarin.

ᴥ Pattern:

(+) Subject + was/were + Verb-ing + Object/Complement(–) Subject + was/were + not + Verb-ing + Object/Complement(?) Was/Were + Subject + Verb-ing + Object/Complement + ?

ᴥ Notes:

Was, digunakan pada subject: I, he, she, dan it. Were, digunakan pada subject: You, we, dan they.

◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌

ʕ Simple Past Future ʔ

ᴥ Fungsi

Mengungkapkan suatu peristiwa/kegiatan yang akan terjadi namun dengan persepsi masa lalu.

ᴥ Pattern:

(+) Subject + would + Verb 1 + Object/Complement(–) Subject + wouldn't + Verb 1 + Object/Complement(?) Would + Subject + Verb 1 + Object/Complement + ?

◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌

ʕ Simple Past Perfect ʔ

ᴥ Fungsi

Mengungkapkan suatu peristiwa/kegiatan yang telah terjadi di masa lalu di mana waktu terjadinya dijelaskan secara spesifik. Misalnya: Aku telah menyelesaikan tugas sekolah ku sebelum Ibu menelpon ku.

ᴥ Pattern:

(+) Subject + had + Verb 3 + Object/Complement(–) Subject + hadn't + Verb 3 + Object/Complement(?) Had + Subject + Verb 3 + Object/Complement + ?

◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌◌

ʕ Simple Past Perfect Continuous ʔ

ᴥ Fungsi

Mengungkapkan suatu peristiwa/kegiatan yang telah terjadi di masa lalu di mana kehiatan tersebut masih berlangsung di masa tersebut.

ᴥ Pattern:

(+) Subject + had + been + Verb-ing + Object/Complement(–) Subject + hadn't + been + Verb-ing + Object/Complement(?) Had + Subject + been + Verb-ing + Object/Complement + ?
