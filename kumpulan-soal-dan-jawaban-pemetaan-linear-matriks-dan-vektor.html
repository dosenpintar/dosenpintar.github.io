6. Matriks yang memetakan titik G(2, -3) menjadi G'(-13, 12) adalah ...​


Salah satu contoh matriks yang memetakan titik G(2, -3) menjadi G'(-13, 12) adalah [tex]\left[\begin{array}{ccc}-5&1\\3&-2\\\end{array}\right].[/tex]

Penjelasan dengan langkah-langkah:

Hubungan antara titik G(x, y) dengan bayangan titik G, yaitu titik G'(x', y'), di mana sebuah matriks [tex]\left[\begin{array}{ccc}a&b\\c&d\\\end{array}\right][/tex] memetakan titik G menjadi G' adalah sebagai berikut.

[tex]\left[\begin{array}{ccc}x'\\y'\\\end{array}\right] = \left[\begin{array}{ccc}a&b\\c&d\\\end{array}\right]\left[\begin{array}{ccc}x\\y\\\end{array}\right][/tex]

Selanjutnya persamaan tersebut diolah menjadi sistem persamaan seperti yang berikut ini.

[tex]x' = ax + by[/tex]

[tex]y' = cx + dy[/tex]

Diketahui:

Titik G(2, -3) dan titik G'(-13, 12).Matriks [tex]\left[\begin{array}{ccc}a&b\\c&d\\\end{array}\right][/tex] yang memetakan titik G menjadi G'

Ditanya:

Matriks [tex]\left[\begin{array}{ccc}a&b\\c&d\\\end{array}\right][/tex].

Proses:

[tex]\left[\begin{array}{ccc}-13\\12\\\end{array}\right] = \left[\begin{array}{ccc}a&b\\c&d\\\end{array}\right]\left[\begin{array}{ccc}2\\-3\\\end{array}\right][/tex]

Terbentuk sistem persamaan, yakni

2a - 3b = -13

2c - 3d = 12

Contoh nilai-nilai a, b, c, dan d yang memenuhi adalah a = -5, b = 1, c = 3, dan d = -2.

Bukti:

2(-5) - 3(1) = -132(3) - 3(-20 = 12

Jadi, salah satu matriks yang memetakan titik G(2, -3) menjadi G'(-13, 12) adalah [tex]\left[\begin{array}{ccc}-5&1\\3&-2\\\end{array}\right].[/tex]

Apabila soal ini berjenis pilihan ganda, maka cara di atas dapat digunakan dalam menentukan matriks yang tepat sesuai opsi yang tersedia. Jadi, matriks-matriks yang tersedia di pilihan jawaban dapat dicoba satu persatu hingga ditemukan sebuah matriks yang sesuai dengan persamaan [tex]\left[\begin{array}{ccc}-13\\12\\\end{array}\right] = \left[\begin{array}{ccc}a&b\\c&d\\\end{array}\right]\left[\begin{array}{ccc}2\\-3\\\end{array}\right].[/tex]

10. diketahui pemetaan a(x,y) dirifleksikan a'(-y,-x). matriks transformasi yang bersesuaian dengan pemetaan tersebut adalah


Matriks transformasi yang sesuai dengan pemetaan A (x,y) direfleksikan menjadi A' (-y,-x) adalah [tex]\left[\begin{array}{ccc}0&-1\\\\-1&0\end{array}\right][/tex].

Pembahasan

Diketahui:

A = (x,y)

A' = (-y,-x)

Ditanya:

matriks transformasi ......... ?

Penyelesaian:

Transformasi geometri secara umum dapat dinyatakan dengan matriks [tex]\left[\begin{array}{ccc}a&b\\\\c&d\end{array}\right][/tex] yang memetakan titik A (x,y) ke titik A' (x',y'). Persamaannya adalah:

[tex]\left[\begin{array}{ccc}x'\\\\y'\end{array}\right] =\left[\begin{array}{ccc}a&b\\\\c&d\end{array}\right] \left[\begin{array}{ccc}x\\\\y\end{array}\right][/tex]

[tex]\left[\begin{array}{ccc}x'\\\\y'\end{array}\right] =\left[\begin{array}{ccc}a.x+b.y\\\\c.x+d.y\end{array}\right][/tex]

Maka soal di atas dapat dicari matriksnya dengan cara:

[tex]\left[\begin{array}{ccc}-y\\\\-x\end{array}\right] =\left[\begin{array}{ccc}a.x+b.y\\\\c.x+d.y\end{array}\right][/tex]

Untuk lebih mudah kita cari masing-masing nilai secara terpisah.

nilai x' = -y dicari dengan cara

-y = a.x + b.y

karena hasilnya hanya nilai y maka nilai a.x = 0 -> a = 0

-y = 0.x + by

-y = 0 + by

b = [tex]\frac{-y}{y}[/tex]

b = -1

nilai y' = -x dicari dengan cara

-x = c.x + d.y

karena hasilnya hanya nilai x maka nilai d.y = 0 -> d = 0

-x = cx + 0.y

-x = cx + 0

c = [tex]\frac{-x}{x}[/tex]

c = -1

Dari perhitungan diperoleh nilai-nilai berikut:

a = 0b = -1c = -1d = 0

Matriks transformasi = [tex]\left[\begin{array}{ccc}a&b\\\\c&d\end{array}\right] = \left[\begin{array}{ccc}0&-1\\\\-1&0\end{array}\right][/tex]

Untuk lebih memahami konsepnya bisa menyimak konsep refleksi dalam transformasi geometri di lampiran, ya!

31. 3. Diketahui : T: P₁ → R³ ->> T(C₁ + C₂t) = (C₂, C₁, C₁ - C₂) a. Tunjukkan T pemetaan linear Tentukan inti pemetaan linear T b. C. Periksa apakah T pemetaan satu-satu atau tidak.​


Jawab:

a. Untuk menunjukkan T sebagai pemetaan linear, perlu dibuktikan bahwa T memenuhi dua sifat pemetaan linear, yaitu:

1. T(x + y) = T(x) + T(y)

2. T(kx) = kT(x)

Misalkan T : P₁ → R³ dengan T(C₁ + C₂t) = (C₂, C₁, C₁ - C₂).

1. Untuk sifat 1, misalkan x = C₁ + C₂t dan y = D₁ + D₂t. Maka,

  T(x + y) = T(C₁ + C₂t + D₁ + D₂t)

           = T((C₁ + D₁) + (C₂ + D₂)t)

           = (C₂ + D₂, C₁ + D₁, C₁ + D₁ - C₂ - D₂)

           = (C₂, C₁, C₁ - C₂) + (D₂, D₁, D₁ - D₂)

           = T(C₁ + C₂t) + T(D₁ + D₂t)

           = T(x) + T(y)

  Sehingga sifat 1 terpenuhi.

2. Untuk sifat 2, misalkan k adalah suatu bilangan skalar dan x = C₁ + C₂t. Maka,

  T(kx) = T(k(C₁ + C₂t))
