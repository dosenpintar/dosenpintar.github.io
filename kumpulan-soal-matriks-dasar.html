Misalkan ada persamaan matriks

[tex] \binom{a \: \: b}{c \: \: d} = \binom{e \: \: f}{g \: \: h} [/tex]

Maka :

a = e

b = f

c = g

d = h

Untuk membuktikan nilai a, b, dan c benar atau tidak, maka substitusikan nilai a, b dan c ke dalam matriks-matriks tersebut. Jika benar, maka kedua matriks tersebut akan bernilai sama.

[tex] \begin{align} R &= S^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&\frac{1}{2}x\\ 4y&-3 \end{pmatrix}^T \\ \begin{pmatrix} 5&2\\4&-3\end{pmatrix} &= \begin{pmatrix} 5&4y\\ \frac{1}{2}x&-3 \end{pmatrix} \end{align} [/tex]

Diperoleh dari kesamaan nilai matriks di atas:

[tex] \begin{align} 2&=4y \\ y &= \dfrac{2}{4} = \dfrac{1}{2} \end{align} [/tex]

kemudian:

[tex] \begin{align} 4 &= \dfrac{1}{2}x \\ x &= 4\cdot 2 = 8 \end{align} [/tex]

Sehingga nilai [tex] x+y = 8 + \frac{1}{2} = 8 \frac{1}{2} = \frac{17}{2} [/tex]

27. Buatlah program untuk menukarkan dua buah matriks 2x4 menggunakan method dan array dengan ketentuan: a. Terdiri dari 2 method: matriks1 dan matriks2 yang nilai kembaliannnya berupa array. b. Nilai matriks1: 3 1 4 2 5 3 Nilai matriks2: 9 4 6 8 7 1


Jawaban:

Berikut jawabannya dalam bahasa C:

#define _CRT_SECURE_NO_WARNINGS

#include<stdio.h>

void printArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

printf("Matrix 1:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix1[i][j]);

 }

 printf("\n");

}

printf("\nMatrix 2:\n");

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  printf("%d ",matrix2[i][j]);

 }

 printf("\n");

}

}

void swapArray(int cols,int rows,int matrix1[3][2],int matrix2[3][2])

{

int temp;

for(int i=0; i<rows; i++)

{

 for(int j=0; j<cols; j++)

 {

  temp = matrix1[i][j];

  matrix1[i][j] = matrix2[i][j];

  matrix2[i][j] = temp;

 }

 

}

}

int main()

{

int cols = 2;

int rows = 3;

int matrix1[3][2] =  

{

 {3,1},

 {4,2},

 {5,3}

};

int matrix2[3][2] =  

{

 {9,4},

 {6,8},

 {7,1}

};

printArray(cols,rows,matrix1,matrix2);

swapArray(cols,rows,matrix1,matrix2);

printf("\nAfter Swap\n\n");

printArray(cols,rows,matrix1,matrix2);

getchar();

return 0;

}

Penjelasan:

Program ini pertama menginisialisasi jumlah baris dan kolom. Kemudian matrix1 dan matrix2 diinisialisasi dengan nilai yang telah ditentukan. Untuk mengetahui perbedaan antara sebelum dan sesudah maka dibuatlah method printArray untuk mencetak matrix1 dan matrix2. Method printArray digunakan 2 kali untuk melihat nilai sebelum dan sesudah. Terakhir ada method untuk swap nilai matrix 1 dan matrix 2 dengan menggunakan bantuan variable temp untuk menyimpan nilai matrix 1 sebelum dioverwrite nilai matrix2 dan digunakan untuk mengoverwrite nilai matrix2.

Diketahui matriks  [tex]A = \left(\begin{array}{ccc}-2&4\\5&1\end{array}\right)[/tex] dan [tex]B = \left(\begin{array}{ccc}1&-2\\3&-2\end{array}\right)[/tex] Matriks hasil dari AB adalah [tex]\left(\begin{array}{ccc}10&-4\\8&-12\end{array}\right)[/tex]

Jika [tex]A = \left(\begin{array}{ccc}2&3\\5&-1\end{array}\right)[/tex] dam [tex]B = \left(\begin{array}{ccc}1\\2\end{array}\right)[/tex] Matriks 2AB adalah [tex]\left(\begin{array}{ccc}16\\6\end{array}\right)[/tex]

Jika [tex]A = \left(\begin{array}{ccc}-2&3\\2&1\end{array}\right)[/tex] dan [tex]B = \left(\begin{array}{ccc}1\\-1\end{array}\right)[/tex]Matriks [tex]AB^T[/tex] adalah [tex]\left(\begin{array}{ccc}-5&1\end{array}\right)[/tex]

PEMBAHASAN :

Matriks merupakan bilangan yang tersusun secara berbaris sedemikian rupa dengan tujuan memudahkan penyampaian data supaya mudah diolah.

Martiks itu sendiri memiliki beberapa rumus yakni :

Jika A + B = [tex]\left[\begin{array}{ccc}1&2\\3&2\end{array}\right] +\left[\begin{array}{ccc}2 & 3\\1 & 4\end{array}\right] = \left[\begin{array}{ccc}(1 +2)&(2+3)\\(3+1)&(2 + 4)\end{array}\right][/tex]

Jika A - B =  [tex]\left[\begin{array}{ccc}1&2\\3&2\end{array}\right] -\left[\begin{array}{ccc}2 & 3\\1 & 4\end{array}\right] = \left[\begin{array}{ccc}(1 -2)&(2-3)\\(3-1)&(2 - 4)\end{array}\right][/tex]

Jika AB    =  [tex]\left[\begin{array}{ccc}1&2\end{array}\right] \left[\begin{array}{ccc}3\\4\end{array}\right] = \left[\begin{array}{ccc}(1.3) &\ (2.4)\end{array}\right][/tex]

Jika [tex]\frac{A}{B}[/tex]       = [tex]A^{-1}B[/tex] (Menggunakan Determine)

Matriks pada penjumlahan dan penguranan juga dapat kita adaptasikan dengan sifat operasi bilangan bulat seperti sifat komutatif, asosiatif, dan distributif, namun matriks memiliki 5 sifat yaitu :

1. A + B = B + A (Sifat Komutatif)

2. (A + B) + C = A + (B + C) (Sifat Asosiatif)

3. A + 0 = 0 + A (Jika sifat matriks 0)

4. A + B = 0 => A = -B (Pindah Ruas)

5. A - B = A + (-B)

Jenis Matriks pada umumnya dibagi menjadi 9 yakni Matriks Baris, Kolom, Persegi, Nol, Segitiga Atas, Segitiga Bawah, Diagonal, Skalar, dan Identitas

Matriks juga memiliki transpose, apa itu transpose? Transpose pada Matriks adalah suatu matriks yang dikerjakan pertukaran antara dimensi kolom dan baris. Contohnya  [tex]\left[\begin{array}{ccc}1&2&3\end{array}\right] =\left[\begin{array}{ccc}1\\2\\3\end{array}\right][/tex] Transpose itu sendiri perpindahannya dengan cara diagonal. Diagonal itu dengan garis seperti ini / setiap kolom. Jika masih tidak paham dapat di lihat di lampiran.
