19. Materi tentang Matematika prisma segitiga kelas 8 semester 2.Tolong bantu nomor 10, 11, dan 12. satu soal juga gpp. Makasih banyak​


Jawaban:

10) K = 123cm

11) V = 600cm³

12) L = 648cm²

Penjelasan dengan langkah-langkah:

Diketahui :

t prisma = 25cm

CD = 8cm

DE = 6cm

Ditanya :

a. panjang kawat = ...?

b. Volume prisma =...?

c. Luas prisma = ...?

Penyelesaian :

10) Sebelum mencari panjang kawat, kita hitung sisi miring segitiga (CE) dengan Teorema Pythagoras

CE² = CD² + DE²

CE² = 8² + 6²

CE² = 64 + 36

CE = _/100

CE = 10cm

untuk mengatahui panjang kawat sebagai kerangaka, maka kita menghitung keliling prisma :

K = (2 x k alas ) + (3 × t prisma)

K = ( 2 × ( CD + DE + CE) ) + ( 3 × 25cm)

K = ( 2 × ( 8cm + 6cm + 10cm)) + 75cm

K = (2 × 24cm) + 75cm

K = 48cm + 75cm

K = 123cm

Maka, panjang kawat yang diperlukan untuk membuat kerangka prisma adalah 123cm.

11) Volume prisma

V = Luas alas × tinggi prisma

V = (½ × CD × DE) × 25cm

V = (½ × 8cm × 6cm) × 25cm

V = 24cm² × 25cm

V = 600cm³

12) Luas permukaan prisma

L = (2 × L alas ) + ( K alas × tinggi prisma)

L = ( 2 × (½ × CD × DE)) + (( CD + DE + CE ) × 25cm))

L = ( 2 × ( ½ × 8cm × 6cm)) + (( 8cm + 6cm + 10cm) × 25cm

L = (2 × 24cm²) + (24cm × 25cm)

L = 48cm² + 600cm²

L = 648cm²
