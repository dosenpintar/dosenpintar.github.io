1. persamaan diferensial eksak​


[tex]Solusi~dari~\frac{dx}{dy}=-\frac{(x+2y)}{y^2+2x}~adalah~\frac{1}{2}x^2+2xy+\frac{1}{3}y^3=C[/tex]

PEMBAHASAN

Persamaan diferensial [tex]Mdx+Ndy=0[/tex] disebut eksak jika memenuhi :

[tex]\frac{\partial{M}}{\partial{y}}=\frac{\partial{N}}{\partial{x}}[/tex]

Solusi dari persamaan diferensial ini adalah [tex]F(x,y)=C[/tex]

Langkah langkah untuk mencari solusi [tex]F(x,y)=C[/tex]

1. Karena [tex]\frac{\partial{F(x,y)}}{\partial{x}}=M(x,y)[/tex] maka [tex]F(x,y)=\int\limits^x {M(x,y)} \, dx+g(y)[/tex]

2. Turunkan F(x,y) terhadap y.

3. Bandingkan hasil no 2 dengan fungsi N untuk memperoleh fungsi g(y).  

.

DIKETAHUI

[tex]\frac{dy}{dx}=-\frac{(x+2y)}{y^2+2x}\\[/tex]

.

DITANYA

Tentukan solusi dari persamaan diferensial eksak tersebut.

.

PENYELESAIAN

[tex]\frac{dy}{dx}=-\frac{(x+2y)}{y^2+2x}\\\\(y^2+2x)dy=-(x+2y)dx\\\\(x+2y)dx+(y^2+2x)dy=0\\\\diperoleh~:\\\\M=x+2y\\\\N=y^2+2x\\\\\\Cek~nilai~\frac{\partial{M}}{\partial{y}}~dan~\frac{\partial{N}}{\partial{x}}\\\\\frac{\partial{M}}{\partial{y}}=2\\\\\frac{\partial{N}}{\partial{x}}=2\\\\Karena~\frac{\partial{M}}{\partial{y}}=\frac{\partial{N}}{\partial{x}}~maka~PD~eksak\\[/tex]

> Cari solusi PD

[tex]F(x,y)=\int\limits^x {M(x,y)} \, dx\\\\F(x,y)=\int\limits^x {x+2y} \, dx\\\\F(x,y)=\frac{1}{2}x^2+2xy+g(y)[/tex]

.

Kita turunkan F(x,y) terhadap y

[tex]\frac{\partial{F(x,y)}}{\partial{y}}=2x+g'(y)\\\\Karena~\frac{\partial{F(x,y)}}{\partial{y}}=N\\\\maka\\\\2x+g'(y)=y^2+2x[/tex]

dengan menyamakan kedua ruas diperoleh

[tex]g'(y)=y^2\\\\g(y)=\int\limits {y^2} \, dy\\\\g(y)=\frac{1}{3}y^3+C[/tex]

Sehingga solusi dari PD eksak tersebut adalah :

[tex]\frac{1}{2}x^2+2xy+g(y)=C\\\\\frac{1}{2}x^2+2xy+\frac{1}{3}y^3=C[/tex]

.

KESIMPULAN

[tex]Solusi~dari~\frac{dx}{dy}=-\frac{(x+2y)}{y^2+2x}~adalah~\frac{1}{2}x^2+2xy+\frac{1}{3}y^3=C[/tex]

.

14. Perhatikan persamaan diferensial berikut. Selidiki apakah persamaan diferensial tersebut merupakan persamaan diferensial eksak!. Jika ya, carilah solusinya. Bila tidak, jelaskan alasanmu. a. 5x + 10y + 10x + 6y dy = 0 b.12x + 7y dx + 13x + 8y dy = 0


Terdapat dua persamaan diferensial (PD) yang tercantum pada soal. Ketika diperiksa, hanya PD berikut yang merupakan PD eksak: (5x+10y)dx+(10x+6y)dy = 0. PD: (12x+7y)dx+(13x+8y)dy = 0 tidak eksak karena  [tex]\bf\frac{\partial M}{\partial y}[/tex] ≠ [tex]\bf\frac{\partial N}{\partial x}[/tex], dengan M = 12x+7y dan N = 13x+8y. Solusi dari PD eksak tersebut adalah ⁵⁄₂x²+10xy+3y² = C.

Penjelasan dengan langkah-langkah

Sepertinya ada kesalahan tulis pada PD yang diberikan. Kedua PD tidak diberikan tanda kurung, seperti yang tertulis pada paragraf pertama di atas. Lalu, pada PD pertama (poin a), tidak ada notasi dx, yang seharusnya ada seperti yang tertulis pada paragraf pertama di atas.

Diketahui:

Dua PD:

a. (5x+10y)dx+(10x+6y)dy = 0

b. (12x+7y)dx+(13x+8y)dy = 0

Ditanya: PD yang merupakan PD eksak, solusi PD eksak, dan alasan PD yang bukan PD eksak

Jawab:

Untuk poin a:

Pemisalan

Misalkan bahwa:

M = 5x+10y

N = 10x+6y

Turunan parsial

[tex]\frac{\partial M}{\partial y}[/tex] = 10

[tex]\frac{\partial N}{\partial x}[/tex] = 10

Penentuan PD eksak

Karena [tex]\frac{\partial M}{\partial y}[/tex] = [tex]\frac{\partial N}{\partial x}[/tex], maka PD ini adalah PD eksak.

Pemisalan fungsi konstan

Misalkan F(x,y) = C₁ yang merupakan fungsi konstan.

Informasi tersirat dari PD

Berdasarkan bentuk PD: (5x+10y)dx+(10x+6y)dy = 0, diketahui:

[tex]\frac{\partial F}{\partial x}[/tex] = 5x+10y...(*)

[tex]\frac{\partial F}{\partial y}[/tex] = 10x+6y...(**)

Integral parsial terhadap x

Persamaan (*) diintegral parsial terhadap x, sehingga diperoleh:

F = ⁵⁄₂x²+10xy+ψ(x,y)

Turunan parsial F terhadap y

[tex]\frac{\partial F}{\partial y}[/tex] = 10x+ψ'(x,y)

Perbandingan turunan parsial dengan persamaan

Dengan persamaan (**) dan turunan yang baru diperoleh sebelumnya, maka:

10x+ψ'(x,y) = 10x+6y

ψ'(x,y) = 6y

ψ(x,y) = 3y²+C₂

Solusi

F = ⁵⁄₂x²+10xy+3y²+C₂ = C₁

⁵⁄₂x²+10xy+3y² = C, dengan C = C₁-C₂

Jadi, solusinya adalah ⁵⁄₂x²+10xy+3y² = C.

Untuk poin b:

Pemisalan

Misalkan bahwa:

M = 12x+7y

N = 13x+8y

Turunan parsial

[tex]\frac{\partial M}{\partial y}[/tex] = 7

18. . Persamaan Diferensial Eksak (3x + 2y) dx + (2x + y) dy = 0 , dengan y(0) = 2​


Jawab:

Untuk menyelesaikan persamaan diferensial eksak, kita perlu memeriksa apakah persamaan tersebut memenuhi kondisi persamaan diferensial eksak. Dalam hal ini, kita akan memeriksa apakah persamaan diferensial dapat diekspresikan dalam bentuk:

M(x, y) dx + N(x, y) dy = 0

di mana ∂M/∂y = ∂N/∂x.

Dalam kasus ini, persamaan diferensial diberikan oleh:

(3x + 2y) dx + (2x + y) dy = 0

Dalam hal ini, kita dapat melihat bahwa ∂M/∂y = 2 dan ∂N/∂x = 2. Karena kedua suku ini sama, persamaan diferensial memenuhi kondisi persamaan diferensial eksak.

Untuk menyelesaikan persamaan ini, kita perlu mencari fungsi ϕ(x, y) sehingga:

∂ϕ/∂x = M(x, y) = 3x + 2y

∂ϕ/∂y = N(x, y) = 2x + y

Mari kita mulai dengan mencari fungsi ϕ(x, y) dengan mengintegralkan persamaan pertama terhadap x:

ϕ(x, y) = ∫ (3x + 2y) dx

ϕ(x, y) = (3/2)x^2 + 2xy + C(y)

Di sini, C(y) adalah fungsi yang hanya tergantung pada y. Sekarang kita akan mencari ∂ϕ/∂y dari hasil ini:

∂ϕ/∂y = 2x + C'(y)

Karena kita tahu bahwa ∂ϕ/∂y = N(x, y), maka kita dapat menyamakan suku-suku ini:

2x + C'(y) = 2x + y

Dari sini, kita dapat melihat bahwa C'(y) = y, yang berarti C(y) = (1/2)y^2 + D, di mana D adalah konstanta integrasi.

Akhirnya, kita dapat menyusun fungsi ϕ(x, y) secara keseluruhan:

ϕ(x, y) = (3/2)x^2 + 2xy + (1/2)y^2 + D

Untuk menentukan nilai D, kita akan menggunakan kondisi awal y(0) = 2:

ϕ(0, 2) = (3/2)(0)^2 + 2(0)(2) + (1/2)(2)^2 + D

ϕ(0, 2) = 0 + 0 + 2 + D = 2 + D

Karena y(0) = 2, maka kita harus memiliki ϕ(0, 2) = 0. Dengan demikian, kita dapat menentukan D:

2 + D = 0

D = -2

Akhirnya, persamaan ϕ(x, y) menjadi:

ϕ(x, y) = (3/2)x^2 + 2xy + (1/2)y^2 - 2

Jadi, solusi persamaan diferensial eksak (3x + 2y) dx + (2x + y) dy = 0 dengan y(0) =

2 adalah ϕ(x, y) = (3/2)x^2 + 2xy + (1/2)y^2 - 2.

Diketahui suatu persamaan diferensial (PD): (6x+5y)dx = -(5x+3y)dy dan y(0) = 2. PD tersebut merupakan PD eksak. Solusi dari PD tersebut adalah 3x²+5xy+³⁄₂y² = 6.

Penjelasan dengan langkah-langkah

Diketahui:

(6x+5y)dx = -(5x+3y)dy

y(0) = 2

Ditanya: PD eksak/tidak? Jika eksak, cari solusinya

Jawab:

PD tersebut dapat ditulis kembali menjadi:

(6x+5y)dx = -(5x+3y)dy

(6x+5y)dx+(5x+3y)dy = 0

Misalkan bahwa:

M = 6x+5y dan N = 5x+3y

Mari tentukan turunan parsial: [tex]\frac{\partial M}{\partial y}[/tex] dan [tex]\frac{\partial N}{\partial x}[/tex].

[tex]\frac{\partial M}{\partial y}=5\\\frac{\partial N}{\partial x}=5[/tex]

[tex]\frac{\partial F}{\partial x}=6x+5y\cdots(1)\\\frac{\partial F}{\partial y}=5x+3y\cdots(2)[/tex]

F = 3x²+5xy+ψ(x,y)

[tex]\frac{\partial F}{\partial y}=5x+\psi'(x,y)[/tex]

ψ'(x,y) = 3y

ψ(x,y) = ³⁄₂y²+C₂

Dengan ini, didapatkan bentuk lengkap dari F sebagai berikut:

F = 3x²+5xy+³⁄₂y²+C₂ = C₁

3x²+5xy+³⁄₂y² = C (dengan C = C₁-C₂)

3·0²+5·0·2+³⁄₂·2² = C

3·0+0+³⁄₂·4 = C

0+6 = C

C = 6

Jadi, solusinya adalah 3x²+5xy+³⁄₂y² = 6.

Persamaan diferensial non eksak [tex](6 y^2+ 4xy^3) dx + (6xy + 4 x^2 y^2) dy = 0[/tex]

Ditanya:

Faktor integrasi dan solusinya!

Pembahasan:

Kita cari terlebih dahulu faktor integrasinya

Misalkan

[tex]M=6 y^2+ 4xy^3\\N=6xy + 4 x^2 y^2[/tex]

[tex]g(x)=\frac{\frac{\partial M}{y} -\frac{\partial N}{x} }{N}\\ g(x)=\frac{12y+12xy^2-(6y+8xy^2)}{6xy + 4 x^2 y^2} \\g(x)=\frac{6y+4xy^2}{6xy+4x^2y^2}\\ g(x)=\frac{6y+4xy^2}{x(6y+4xy^2)}\\g(x)=\frac{1}{x}[/tex]

Sehingga faktor integrasinya adalah:

[tex]\mu=e^{\int g(x) dx}=e^{\int \frac{1}{x} dx}=x[/tex]

Maka PD dapat diubah menjadi eksak

[tex]x (6 y^2+ 4xy^3) dx+x(6xy + 4 x^2 y^2) dy =0\\(6xy^2+4x^2y^3)dx+(6x^2y+4x^3y^2)dy=0[/tex]

[tex]u=\int (6xy^2+4x^2y^3) dx + k(y)\\u=3x^2y^2+\frac{4x^3y^3}{3} +k(y)[/tex]

[tex]\frac{\partial u}{\partial y} =6x^2y+4x^3y^2+\frac{dk}{dy} =N\\6x^2y+4x^3y^2+\frac{dk}{dy} =6xy + 4 x^2 y^2\\\frac{dk}{dy}=6xy + 4 x^2 y^2-6x^2y-4x^3y^2\\k(y)=3xy^2+\frac{4x^2y^3}{3}-3x^2y^2 -\frac{4x^3y^3}{3}[/tex]

Sehingga solusinya adalah

[tex]u=3x^2y^2+\frac{4x^3y^3}{3}+3xy^2+\frac{4x^2y^3}{3}-3x^2y^2 -\frac{4x^3y^3}{3}=C\\ 3xy^2+\frac{4x^2y^3}{3}=C[/tex]
