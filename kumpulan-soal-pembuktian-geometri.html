2. Buktikan identitas geometri ini..


nomor 1
(1+ sin x)(1-sin x)
= 1 -sin x + sin x + sin² x
= 1 - sin² x (ingat : sin² x + cos² x = 1 )
= cos² x (terbukti)

nomor 2
( sin x - cos x )²
= (sin x - cos x)(sin x - cos x)
= sin² x - sin x cos x - sin x cos x + cos² x
= ( sin² x + cos² x ) - 2 sin x cos x
= 1 - 2 sin x cos x

nomor 3
3 cos² x
= 3 ( 1 - sin² x )
= 3 - 3 sin² x

nomor 4
( sin x + cos x )( sin x + cos x )
= sin² x + 2 sin x cos x + cos² x
= 1 + 2 sin x cos x (gak bisa nglanjutin,hehe)

nomor 5
(sin x - cos x)² + 2 cos x sin x
= sin² x + cos² x - 2 cos x sin x + 2 cos x sin x
= sin² x + cos² x
= 1

13. buatlah rangkuman materi barisan dan deret geometripliss di jawab KK,soalnya mau di kumpul sekarang?​


Jawaban:

A. Barisan Geometri

Barisan geometri adalah baris yang nilai setiap sukunya didapatkan dari suku sebelumnya melalui perkalian dengan suatu bilangan r/rasio

Rumus barisan geometri:

Un = a*r^(n-1)

Un = nilai suku ke-n

a/U1 = nilai suku pertama

r = perbandingan/rasio antara nilai suku ke-n dengan nilai suku sebelumnya (n-1) yang berdekatan selalu sama (rumus r: Un/Un-1)

n = suku ke-n

Untuk mencari nilai suku ke-n jika tak diketahui nilai a dari suatu barisan geometri dengan mengetahui nilai suku ke-k dan r dengan rumus:

Un = Uk*r^(n-k)

B. Deret Geometri

Deret geometri adalah penjumlahan suku-suku pertama sampai suku ke-n suatu barisan geometri.

Rumus deret geometri:

Sn = U1+U2+U3+...+U(n-1)+Un atau

Sn = a+ar+ar²+...+ar^(n-2)+ar^(n-1)

Jika hanya diketahui nilai a dan nilai r, maka rumusnya adalah:

Sn = a(1-r^n)/1-r jika 0 < r < 1

Sn = a(r^n-1)/r-1 jika r > 1

22. Diketahui sudut geometri datar : <2 komplemen <1 kompelemen <1 = Buktikan :


Penjelasan dengan langkah-langkah:

Untuk membuktikan bahwa sudut geometri datar yang diketahui memenuhi kondisi yang diberikan, kita perlu mengetahui rumus-rumus yang digunakan dalam geometri datar. Pertama-tama, kita perlu tahu bahwa komplemen dari suatu sudut adalah sudut yang membentuk sudut lengkung dengan sudut tersebut sebesar 90 derajat. Jadi, jika suatu sudut memiliki komplemen, maka jumlah sudut tersebut dengan komplemennya adalah 90 derajat.

Selanjutnya, kita juga perlu tahu bahwa suatu sudut yang memiliki komplemen disebut sudut tepat. Jadi, jika suatu sudut memiliki komplemen, maka sudut tersebut adalah sudut tepat.

Berdasarkan kedua informasi tersebut, kita dapat membuktikan bahwa sudut geometri datar yang diketahui memenuhi kondisi yang diberikan sebagai berikut:

Sudut <2 memiliki komplemen <1 komplemen <1. Ini berarti bahwa sudut <1 komplemen <1 adalah komplemen dari komplemen sudut <2. Dengan kata lain, jumlah sudut <2 dan komplemen komplemen <1 adalah 90 derajat.

Sudut <1 komplemen <1 adalah sudut tepat. Ini berarti bahwa sudut <1 komplemen <1 memiliki komplemen.

Dengan demikian, sudut geometri datar yang diketahui memenuhi kondisi yang diberikan, yaitu sudut <2 memiliki komplemen <1 komplemen <1 dan sudut <1 komplemen <1 adalah sudut tepat.

deret geometri:

U₅ = 1/3

r = 1/3

ditanya:

jumlah tak hingga ...?

jawab:

[tex]U_n =ar^{n - 1}[/tex]

U₅ = ar⁵⁻¹

1/3 = a(1/3)⁴

1 = a(1/3)³

1 = a(1/27)

a = 27

Jumlah deret tak hingga:

[tex]S_{\infty} = \frac{a}{1 - r}[/tex]

[tex]S_{\infty} = \frac{27}{1 - \frac{1}{3} }[/tex]

     [tex]= \frac{27}{\frac{2}{3} }[/tex]

     = 27 x 3/2

     = 81/2

Jadi, jumlah tak hingga deret  geometri tersebut adalah 81/2

26. buktikan bahwa barisan yang di nyatakan oleh un=2^n adalah sebuah barisan geometri​


Un = 2ⁿ

Uji n

Untuk n = 1: U1 = 2¹ = 2Untuk n = 2: U2 = 2² = 4Untuk n = 3: U3 = 2³ = 8Untuk n = 4: U4 = 2⁴ = 16

Uji rasio

U2/U1 = 4/2 = 2U3/U2 = 8/4 = 2U4/U3 = 16/8 = 2

Dari pengujian nilai n diatas, dapat kita simpulkan bahawa rumus Un = 2ⁿ merupakan barisan geometri. Hal ini dapat kita lihat dari rasio yang dibentuk oleh suku-suku tersebut sama.
