40° + 7x = 180°

7x = 140°

80° + 2y = 7x

80° + 2y = 140°

4z + 2y + 40° = 180°

4z + 100° = 180°

4z = 180° - 100°

4z = 80°

z = 80°/4

z = 20°

= 70° (A.)

2y + 10 = 42

2y = 32

x = 26

Jadi, soal matematika ada 26 butir dan ipa ada 16 butir.

39. Gendhis mengikuti lomba Matematika MGMP dengan mengerjakan soal pilihan ganda sebanyak 50 soal. Perhitungan nilai lomba tersebut menggunakan aturan berikut : Jika jawaban benar, maka nilainya 4 Jika jawaban salah, maka nilainya -2 Jika tidak dijawab, maka nilainya 0 Pada saat lomba, Gendhis mampu mengerjakan 42 soal, dan 8 soal diantaranya salah. Berapa nilai yang didapat Gendhis saat mengikuti lomba Matematika MGMP ?​


Diketahui : - mengerjakan 42 soal

- salah 8

- benar 34 ( 42 - 8 )

- tidak mengerjakan 8 soal ( 50-42 )

Ditanya : Berapa nilai yg didapat gendhis saat mengikuti lomba matematika MGMP?

Dijawab : benar : 34 × 4 = 136 point

salah : 8 × (-2) = -16 point

tdk d jwb : 8 × 0 = 0 pint

Jumlah:

136 point - (- 16 point ) + 0 point

= 120 point

Jadi point yg di dapat gendhis saat mengikuti lomba matematika MGMP adalah 120 point
