2. Apa yg di maksud dengan software basis data? sebutkan fungsi fungsinya


Perangkat lunak basis data adalah sebuah perangkat lunak yang digunakan untuk membuat dan mengelola struktur database (basis data) serta untuk mengakses data. Perangkat lunak basis data juga biasa disebut dengan sistem manajemen database.

Fungsinya :
- Merupakan salah satu komponen penting dalam sebuah sistim informasi, merupakan dasar dalam menyediakan informasi dari sebuah data base

- Menentukan kualitas informasi dengan tepat waktu dan relevan serta akurat.

- Mengurangi data ganda/dobel (data redudancy).

- Mengaplikasikan basis data, hubungan antar data dapat lebih meningkat dengan baik.

- Mengurangi pemborosan tempat untuk penyimpanan data di luar.

18. Jelaskan fungsi perangkat lunak basis data?


Perangkat lunak atau software adalah perangkat yang tidak berbentuk beda, namun suatu program atau lainya

Pembahasan

Fungsi perangkat lunak basis data:

Integrasi data

Data yang tersimpan dalam satu tempat, dan satu kali dalam database. Di suatu cabang pertama jadi server pusat, data sudah tercatat, maka jika mencari data tersebut, dari suatu cabang lain. Bisa meminta cabang yang pertama untuk mengirim data tersebut

Membuat integritas data meningkat

Kesatuan data yang lebih akurat, konsisten dan terbaru, kini software bawaan membantu seperti proses tersebut. Salah satu contohnya adalah 'garbage out'.

Untuk meningkatkan keamanan data

Walaupun database bisa diakses melalui komputer, namun keamanan data tetap terlindung, komputer penempatan database atau client tidak bisa menghapus ataupun merubah tanpa izin

28. Jelaskan fungsi perangkat lunak basis data?


Basis data atau data base adalah kumpulan informasi, serta data yang disimpan didalam suatu server komputer secara sistematis, dan diperiksa dan dirubah melalui komputer.

Pembahasan

Fungsi perangkat lunak basis data:

Meningkatkan untuk memelihara data

Perangkat lunak berbasis data dirancang dengan prosedur standar, biasanya bisa juga untuk menambah, mengedit, menghapus data dan lainya.

Dapat menghindari duplikasi data

Sering terjadi suatu hack, maka dari dibuatlah perangkat lunak basis data. Untuk menjaga keamanan data dari duplikasi, serta dapat juga mengatur serta memantau suatu data.

Menjamin kualitas data dan informasi dapat diakses  

Agar jaga tetap aman adalah tugas perangkat lunak, namun juga memberi keleluasaan mengunjungi suaut data dengan aman dan terjaga.

Mendukung kinerja aplikasi yang memerlukan penyimpanan data

Karena sudah banyak sistem penyimpanan data yang membutuhkan software, karena manfaatnya serta kegunaanya dalam menyimpan, serta menjaga data.

Pengelompokan data

Mempermudah untuk mengumpulkan jenis-jenis data, agar mudah dicari secara sejenis atau pengelompokkan

2. Basis Data Elektronik

Jadi basis data ini adalah kumpulan data yang terintegrasi dan tersimpan pada penyimpanan elektronik. Prinsip yang digunakan basis data adalah pengaturan, dimana data yang masuk dalam penyimpanan elektronik tersebut akan diatur didalamnya. Mulai dari nama, jenis data,dan data lainnya akan dikelompokkan dan diurutkan seperti dalam rak buku. Adanya basis data ini dapat mempercepat dan memudahkan user saat pencarian data, seperti yang telah dijelaskan diatas kemudahan dan kecepatan tersebut karena data telah diurutkan dan dikelompokkan. Contoh : Flashdisk, Hard disk, dll.

Untuk data yang tersimpan didalam komputer tentunya perlu sesuatu yang mengatur data tersebut. DBMS (Data Base Manajement System) merupakan sebuah program aplikasi (software) yang fungsinya mengatur data mulai dari mengambil, menghapus,hingga memasukan data. Software yang digunakan adalah Ms. Excel, Ms. Acces,Sql, Oracle,dB2, dBase III, IV, Fox Base.

2. Hirarki merupakan unsur – unsur dari suatu database yang meliputi kumpulan file (berkas), bit, record (cantuman), byte, field (ruas). Hirarki juga di sebut sebagai model pohon karena membentuk seperti struktur pohon. Hirarki kategori dapat mengelompokkan data – data, sehingga menjadi kompleks. Untuk hirarki penyimpanan data terdiri dari beberapa tingkatan data yang di simpan seperti : file, database, bit, byte (karakter), record dan field.

3. Basis data dapat didefinisikan atau diartikan sebagai kumpulan data yang disimpan secara sistematis di dalam komputer yang dapat diolah atau dimanipulasi menggunakan perangkat lunak (software) program atau aplikasi untuk menghasilkan informasi

4. Tujuan utama basis data merupakan untuk dapat memudahkan kita dalam memperoleh data yang kita cari. Selain itu ada beberapa tujuan basis data sebagai berikut yakni : Kecepatan ataupun kemudahan dalam menyimpan, mengedit ataupun memanipulasi data serta dapat melihat data.

5. Sistem manajemen basis data (Bahasa Inggris: database management system, DBMS), atau kadang disingkat SMBD, adalah suatu sistem atau perangkat lunak yang dirancang untuk mengelola suatu basis data dan menjalankan operasi terhadap data yang diminta banyak pengguna.

35. apa yang dimaksud dengan softwere basis data sebutkan fungsi fungsinya


Bisa dikatakan database adalah kumpulan data yang terhubung dalam suatu object (buku telepon, kamus,dll), topik (katalog), atau tujuan khusus tertentu.
Basis Data
Pengertian basis data pada dasarnya terdiri atas 2 kata, yakni “basis “ dan “data”, basis bisa diartikan sebagai dasar, market, tempat atau gudang, sedangkan “data” atau nilai/value berupa fakta yang merepresentasikan deskripsi dari suatu objek atau kejadian, dalam jumlah banyak, kumpulan data ini disebut “informasi”. Misalkan sebuah profil orang adalah informasi orang tersebut, di dalam profil tersebut tercantum data tempat tanggal lahir, alamat tempat tinggal, pekerjaan, dll. Gabungan data tersebut menjadi sebuah informasi mengenai orang bersangkutan.
Keberadaan basis data dalam kuantitas besar dapat kita temukan di data base mahasiswa, SIM, data Kependudukan, di mana jumlah basis data di dalam server itu mencapai ratusan juta informasi. Tentunya akan sulit untuk mencari profil seseorang diantara ratusan juga basis data tanpa bantuan sebuah aplikasi yang dapat men-scan data secara cepat dan akurat.
DBMS
Software Basis Data atau dikenal sebagai Database Management System (DBMS) memungkinkan user melakukan pencarian informasi pada dalam hitungan detik, bahkan bisa kurang tergantung dari hardwarenya, sedangkan di perpustakaan pencarian sebuah buku memakan waktu beberapa menit.
Dewasa ini, database komputer digunakan secara luas di berbagai sektor dan bidang pekerjaan, aplikasi ini menjadi tulang punggung perusahaan atau lembaga dalam pengolahan informasi dan managemen. Secara umum fungsi software basis data adalah sebagai berikut :
1. Merupakan salah satu komponen penting dalam sebuah sistim informasi, merupakan dasar dalam menyediakan informasi dari sebuah data base
2. Menentukan kualitas informasi dengan tepat waktu dan relevan serta akurat.
3. Mengurangi data ganda/dobel (data redudancy).
4. Mengaplikasikan basis data, hubungan antar data dapat lebih meningkat dengan baik.
5. Mengurangi pemborosan tempat untuk penyimpanan data di luar.
